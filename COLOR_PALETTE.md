## 📐 Guía de Uso

### Botones Primarios
```css
background: #467e03;
color: #ffffff;
hover: #3d6f03;
active: #335c02;
```

### Botones Secundarios
```css
background: #e3e6df;
color: #3f3f39;
hover: #c9cdc4;
```

### Links
```css
default: #467e03;
hover: #3d6f03;
visited: #335c02;
```

### Textos
```css
primary: #1a1a18;
secondary: #3f3f39;
tertiary: #707066;
muted: #9fa499;
```

### Fondos
```css
primary: #ffffff;
secondary: #f8f9f7;
tertiary: #f0f2ed;
```

### Bordes
```css
light: #e3e6df;
medium: #c9cdc4;
dark: #9fa499;
```

## 🎨 Combinaciones Recomendadas

### Tarjetas/Cards
- **Fondo**: `#ffffff`
- **Borde**: `#e3e6df`
- **Título**: `#1a1a18`
- **Texto**: `#3f3f39`
- **Badge**: `#467e03` con texto blanco

### Headers
- **Fondo**: `#ffffff`
- **Borde inferior**: `#e3e6df`
- **Logo/Título**: `#1a1a18`
- **Links activos**: `#467e03`
- **Links normales**: `#707066`

### Formularios
- **Input fondo**: `#ffffff`
- **Input borde**: `#e3e6df`
- **Input focus**: `#467e03`
- **Label**: `#3f3f39`
- **Placeholder**: `#9fa499`

### Badges/Tags
- **Verde primario**: Fondo `#467e03`, texto `#ffffff`
- **Verde claro**: Fondo `#e3efd4`, texto `#335c02`
- **Neutral**: Fondo `#e3e6df`, texto `#3f3f39`

## 🌓 Modo Oscuro (Futuro)

Para cuando implementes modo oscuro:

```css
background-primary: #1a1a18;
background-secondary: #2a2a26;
text-primary: #f8f9f7;
text-secondary: #c9cdc4;
primary-green: #89b554; /* Más claro para contraste */
```

---

## ♿ Accesibilidad

Todos los colores cumplen con **WCAG 2.1 AA** para contraste:

✅ **Verde principal (#467e03) sobre blanco**: Ratio 7.8:1  
✅ **Texto principal (#1a1a18) sobre blanco**: Ratio 16.5:1  
✅ **Texto secundario (#3f3f39) sobre blanco**: Ratio 11.2:1

## 🔧 Implementación en el Código

### TypeScript
```typescript
import { colors } from '@/styles/colors';

// Usar colores
backgroundColor: colors.primary[500]
color: colors.text.primary
```

### CSS Variables
```css
/* Disponible en toda la app */
background-color: var(--color-primary);
color: var(--text-primary);
border: 1px solid var(--color-gray-200);
```
