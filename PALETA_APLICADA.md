## 💡 Ventajas de la Nueva Paleta

1. ✅ **Formal y Profesional**: Los tonos verdes oliva transmiten elegancia
2. ✅ **Natural**: Asociado con naturaleza, crecimiento, estabilidad
3. ✅ **Único**: Se diferencia de las paletas azules comunes
4. ✅ **Armonioso**: Grises con tinte verde mantienen cohesión
5. ✅ **Accesible**: Contraste WCAG AA compliant
6. ✅ **Versátil**: Funciona para fotografía de moda y editorial

## 🚀 Cómo Usar los Colores en Nuevos Componentes

### Opción 1: Inline (como está ahora)
```tsx
<button style={{ backgroundColor: '#467e03', color: 'white' }}>
  Click
</button>
```

### Opción 2: Variables CSS
```tsx
<button style={{ backgroundColor: 'var(--color-primary)' }}>
  Click
</button>
```

### Opción 3: Importar desde colors.ts
```tsx
import { colors } from '@/styles/colors';

<button style={{ backgroundColor: colors.primary[500] }}>
  Click
</button>
```

## 📱 Responsive y Estados

### Hover States
Todos los elementos interactivos usan `#3d6f03` (un tono más oscuro) al hacer hover

### Focus States
Los inputs y elementos focuseables usan borde `#467e03` de 2px

### Active States
Los botones presionados usan `#335c02` (aún más oscuro)

### Disabled States
Los elementos deshabilitados usan `#a8ca7e` con opacidad reducida
